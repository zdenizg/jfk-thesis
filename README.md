# JFK OCR Thesis Materials

This repository contains tools and results for extracting text from a set of 55
PDF documents released as part of the JFK 2025 records release. The primary
script uses Google Cloud Vision to perform OCR on the missing PDFs and is
intended for reproducible research.

## Repository Contents

- `ocr_missing_google.py` – Python script that reads `missing_file_ids.txt`,
  converts PDF pages to optimized JPEGs, and submits them to the Vision API
  with retry/backoff and checkpointing.
- `missing_file_ids.txt` – List of 55 document IDs (one per line) corresponding
  to files in `jfk_pdfs/`.
- `ocr_missing_output_google/` – Output directory containing the extracted
  text files (one `.txt` per document) and a `summary.csv`.
- `requirements.txt` – Python dependencies needed to run the script.

> **Note:** Google Cloud credentials are **not** included. Set
> `GOOGLE_APPLICATION_CREDENTIALS` to a service account JSON file before running.

## How it Works

1. Read the list of missing IDs from `missing_file_ids.txt`.
2. For each ID, locate `jfk_pdfs/<ID>.pdf` and convert its pages to images at
   `PDF_DPI` (200 by default).
3. Optimize each page: resize if larger than 2000px, convert to JPEG quality
   75.
4. Send each page to the Vision API's `document_text_detection` method, with a
   45‑second timeout.
5. Retry transient failures (503, 408, rate limits, etc.) up to 8 times using
   exponential backoff and jitter, tracking retry counts.
6. Persist progress per page in `ocr_missing_output_google/progress.json` so an
   interrupted run resumes quickly.
7. Produce a summary CSV and per-document statistics after processing.
8. Final outputs (55 text files) are stored in `ocr_missing_output_google/`.

## Requirements

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

`requirements.txt` contains:

```
google-cloud-vision
pdf2image
pillow
```

You also need [poppler](https://poppler.freedesktop.org/) installed for
`pdf2image`.

## Running the Script

```bash
export GOOGLE_APPLICATION_CREDENTIALS=/path/to/credentials.json
python ocr_missing_google.py
```

The script prints progress and writes output under
`ocr_missing_output_google/`. If you interrupt it, rerunning will resume from
where it left off.
